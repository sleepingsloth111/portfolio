Project: Responsive Redesign

Goals

For my React app, I wanted to practice the following React skills

  • Use components
  • Sort through a list
  • Filter through a list
  • Handle combinations of all of the above

Overview

I decided to make an app for seeing and maintaining a list of Ghibli movies. You can sort by runtime and release date and filter by director and runtime. You can favorite and unfavorite films, where you will see the titles in a favorited box with the average runtime of the favorited films.

Check out my App Here!

Reflection

I left finishing this project much more confident in React. I dabbled in React before in another course, but I was never really confident in it. It was really satisfying to finally understand thoroughly about props, components, and states!
